Sender IDs

This page is the library of reusable Sender ID values. To allow a Sender ID for API sends, attach it to a client or carrier from that record's detail page (Sender IDs tab).

How to associate a Sender ID
  • Clients: Clients → open a client → Sender IDs tab → choose a library entry and Add.
  • Carriers: Carriers → open a carrier → Sender IDs tab (when policy is list, the list is required) → Add.
The columns "Used by Clients" and "Used by Carriers" count how many links exist in the database for each library entry.
